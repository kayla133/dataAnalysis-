# Formula One Race Wins Data

This program shows data about every Formula One race and the winners of that race.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open the code in your favorite editor.
2. Downlad Pandas if not already done.
3. Run the code.

Instructions for using the software:

1. Run the program.
2. Watch the data print in the console.
3. Read the data that you would like to see.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Download your favorite IDE
* Install Pandas
* Make sure that you also included the f1_data.csv file
* Run the program
